# Shellfyre

Comp304 project which similates an Unix-style operating system shell.

## Compile & Run

```bash
gcc shellfyre.c -o shellfyre
```

```bash
./shellfyre
```

## Custom Commands

### filesearch

```bash
shellfyre$ filesearch 
```

### cdh

```bash
shellfyre$ cdh 
```

```bash
shellfyre$ cdh -r
```

### take
```bash
shellfyre$ take
```
### joker

```bash
shellfyre$ joker
```
